Chat Application – User Guide

This guide explains how to use the Chat Application and describes the features currently available. The platform allows users to communicate in real time, customize their profile appearance, and interact using message reactions and emojis.

1. Signing In

To begin using the chat platform, users must sign in with their Google account.

  • Open the login page.
  • Select Login with Google.
  • Choose your Google account.
  • You will automatically be redirected to the chat interface.

After signing in for the first time, your account is created automatically within the system.

2. User List

The left panel of the chat interface displays a list of available users.

  • Each user entry includes their display name and profile picture.
  • Select a user to start a private conversation.
  • The chat window will update to display the conversation.

3. Sending Messages

Messages can be sent instantly and appear in real time for both participants.

  • Type your message in the message input field.
  • Press Enter or click the Send button.
  • Your message will immediately appear in the chat.

4. Emoji Support

The application includes an emoji picker for enhancing messages with expressive icons.

  • Select the emoji button beside the message input.
  • Choose an emoji from the emoji picker.
  • The emoji will be inserted into your message.

5. Message Reactions

Users can react to messages using quick emoji reactions.

Available reactions include:
  • 👍 Like
  • ❤️ Love
  • 😂 Laugh
  • 😮 Surprise
  • 😢 Sad

Reactions appear below the message and update instantly for all participants.

6. Profile Picture Customization

Users can personalize their profile appearance by uploading a custom profile picture.

  • Click the Upload button in the chat interface.
  • Select an image or MP4 video file.
  • Your profile picture will update automatically.

Supported formats include:

  • PNG
  • JPG / JPEG
  • MP4 (looping video avatars)

Your profile picture will be visible:

  • Next to your messages
  • In the user list
  • Beside the profile upload button

7. Real-Time Communication

The chat platform operates in real time. This means:

  • Messages appear instantly.
  • Reactions update immediately.
  • Profile changes are visible without refreshing the page.

8. Current Features

The following features are currently implemented within the application:

  • Google account authentication
  • Real-time private messaging
  • Emoji picker
  • Message reactions
  • Custom profile pictures
  • Video profile avatars (MP4)
  • Live user list

9. Troubleshooting

If you encounter issues while using the application:

  • Refresh the page.
  • Ensure your internet connection is stable.
  • Sign out and sign in again if necessary.
