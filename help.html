# Chat App – Help & User Guide ## Welcome 👋 This page explains how to use the chat application and the features available. The app allows users to sign in with Google, chat with other users in real-time, use emoji reactions, and customize their profile picture. --- # 1. Signing In To start using the chat: 1. Open the homepage. 2. Click **"Login with Google"**. 3. Choose your Google account. 4. You will be redirected to the chat page. Once signed in, your account is automatically created. Your name and email are saved for the chat system. --- # 2. User List On the left side of the chat page you will see a **list of users**. Each user shows: • Their **profile picture** • Their **display name** To start chatting: 1. Click on a user in the list. 2. A chat will open with that user. 3. Messages will appear in real time. --- # 3. Sending Messages 💬 To send a message: 1. Click inside the **message input box**. 2. Type your message. 3. Press **Enter** or click the **Send button**. Your message will instantly appear in the chat. Messages are synchronized in real-time so both users see them immediately. --- # 4. Emoji Picker 😀🎉 You can add emojis to your messages. Steps: 1. Click the **emoji button** beside the message box. 2. The emoji picker will open. 3. Click any emoji to insert it into your message. 4. Send the message normally. --- # 5. Message Reactions 👍 Every message has reaction buttons. You can react with: 👍 Like ❤️ Love 😂 Laugh 😮 Surprise 😢 Sad To react: 1. Hover over a message. 2. Click a reaction emoji. 3. The reaction will appear below the message. Multiple users can react to the same message. --- # 6. Profile Picture You can change your profile picture. Supported formats: • PNG • JPG • JPEG • MP4 video avatars Steps: 1. Click the **Upload button**. 2. Select an image or MP4 video. 3. Your profile picture will update instantly. Your avatar will appear: • Next to the upload button • In the user list • Beside every message you send If you upload a **MP4 video**, it will automatically **loop as your profile picture**. --- # 7. Real-Time Chat This app uses a real-time database. This means: • Messages appear instantly • Reactions update immediately • Profile changes update automatically No page refresh is required. --- # 8. Safety Notes Please follow these guidelines: • Do not share personal or sensitive information • Be respectful to other users • Do not upload inappropriate content --- # 9. Troubleshooting If something does not work: • Refresh the page • Make sure you are logged in • Check your internet connection If the problem continues, try signing out and logging in again. --- # 10. Features Currently Available 🚀 Current features include: ✔ Google Login ✔ Real-time messaging ✔ Emoji picker ✔ Message reactions ✔ Custom profile pictures ✔ Video avatars (MP4) ✔ User list with avatars --- # Thank You For Using The Chat App Enjoy chatting! 🎉
