Chat Application – Help Guide

This guide explains how to use the Chat Application and describes the features currently available. The system allows users to communicate in real time, customize their profile avatar, and view user activity status.

1. Signing In

To begin using the application:

  • Open the login page.
  • Click Login with Google.
  • Select your Google account.
  • You will be redirected to the chat interface.

Your account is automatically created the first time you log in.

2. User List

The left panel displays all users available in the chat system.

  • Each user has a profile avatar.
  • You can see whether a user is Online.
  • If offline, their Last Seen time is shown.
  • Click a user to start a conversation.

3. Sending Messages

Messages are delivered instantly between users.

  • Type your message in the input field.
  • Press Enter or click Send.
  • Your message will appear in the chat immediately.
Tip: Pressing Enter is the fastest way to send messages.

4. Emoji Support

You can add emojis to messages.

  • Click the 😊 Emoji button.
  • Select an emoji from the picker.
  • The emoji will be added to your message.

5. Profile Avatar

You can upload a custom avatar.

  • Click the Upload button.
  • Select an image or MP4 file.
  • Your avatar updates instantly.

Supported formats:

  • PNG
  • JPG / JPEG
  • MP4 (video avatars)

6. Nicknames

You can assign custom nicknames to users.

  • Double-click a user in the user list.
  • Enter the nickname.
  • The nickname will replace the original name for you.
Nicknames are visible only to you.

7. Dark Mode

The chat application includes a dark theme.

  • Click the 🌙 button in the chat header.
  • The interface switches between light and dark mode.
  • Your preference is saved automatically.

8. Real-Time Updates

The chat system updates automatically.

  • Messages appear instantly.
  • User status updates in real time.
  • Profile changes appear immediately.

9. Current Features

  • Google authentication login
  • Private real-time messaging
  • Online / last seen status
  • Emoji picker
  • Custom avatars
  • Video avatars
  • User nickname system
  • Dark mode support
  • Instant message delivery
  • Automatic chat scrolling

10. Troubleshooting

  • Refresh the page if messages stop updating.
  • Check your internet connection.
  • Sign out and sign in again.
